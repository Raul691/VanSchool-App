# 🚐 VanSchool App

Aplicativo mobile desenvolvido como Trabalho de Conclusão de Curso (TCC), com foco em melhorar a experiência e segurança no transporte escolar.

---

## 📖 Sobre o Projeto

O **VanSchool** é um aplicativo mobile criado para conectar responsáveis e motoristas de transporte escolar em uma única plataforma.

A proposta do sistema é trazer mais segurança, praticidade e transparência no acompanhamento de alunos durante o transporte escolar.

Com o aplicativo, responsáveis podem ter mais tranquilidade no dia a dia, enquanto motoristas conseguem melhorar a organização do serviço e a comunicação com os clientes.

---

## 🎯 Objetivo

O objetivo do VanSchool é atuar como uma ponte entre responsáveis e motoristas, permitindo o acompanhamento mais seguro e eficiente do transporte escolar.

### 👨‍👩‍👧 Para responsáveis:

* Mais segurança no transporte das crianças;
* Maior tranquilidade no dia a dia;
* Melhor comunicação com motoristas;
* Acompanhamento mais confiável do serviço.

### 🚐 Para motoristas:

* Melhor organização do trabalho;
* Comunicação facilitada com clientes;
* Mais oportunidades de prestação de serviço;
* Gestão mais eficiente do transporte escolar.

## 🚀 Como Executar o Projeto

### 📌 Pré-requisitos

* Node.js instalado
* App **Expo Go** instalado no celular

### 📥 Instalação

Clone o repositório:

```bash
git clone https://github.com/Raul691/VanSchool_.git
```

Entre na pasta:

```bash
cd VanSchool_
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npx expo start
```

### 📱 Execução no celular

* Abra o aplicativo **Expo Go**
* Escaneie o QR Code exibido no terminal
* O app será carregado automaticamente
